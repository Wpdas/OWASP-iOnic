Funcionalidade Externa Acessar Dados do Aplicativo - Root

Evitar

Deixar dados sensiveis desprotegidos na classe da aplicação ou em algum banco de dados local.


Recomendado

Usar chave segura das distribuidoras de software como Google(Android). Desta forma, as informações sensiveis como de compra por exemplo, não poderam ser acessadas ou computadas/processadas porque a chave da transação será inválida. Um aparelho com Root tem o poder de expor o pacote de uma aplicação, uma vez exposta, quase tudo é possível (Android).
