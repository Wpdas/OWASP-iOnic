Usar áreas de armazenamento públicas como Cookie, LocalStorage, Cartão SD e semelhantes.


Recomendado

Para armazenamento local, use o armazenamento seguro nativo de cada sistema (Secure Storage). Este armazenamento é tratado diretamente com o sistema do dispositivo e somente a aplicação e o SO podem ter acesso.
Observação: Também pode ser usado o Secure Storage.
